# CatlabPing 🐾

**安卓端伴侣查岗 App**，配合 [AstrBot](https://github.com/Soulter/AstrBot) 插件使用。

Cat + Caleb + Ping = CatlabPing

## ✨ 功能模块

- 📍 **位置查岗** — 定时上报 GPS 坐标，配合 [astrbot_plugin_location_guard](https://github.com/Akusative/astrbot_plugin_location_guard)
- 📱 **手机使用监控** — 已安装应用勾选 + App 使用时长上报
- 📸 **概率截屏** — 随机截屏上报，配合 [astrbot_plugin_screenshot_guard](https://github.com/Akusative/astrbot_plugin_screenshot_guard)
- 🔔 **Bark 推送** — 支持 App 端接收推送通知

## 📦 安装

从 [Releases](https://github.com/Akusative/CatlabPing/releases) 下载最新 APK 安装。

## 📱 安卓权限设置（重要）

### 小米 / Redmi (HyperOS)
- 设置 → 应用管理 → CatlabPing → 省电策略 → **无限制**
- 设置 → 应用管理 → CatlabPing → 自启动 → **开启**

### OPPO / realme (ColorOS)
- 设置 → 电池 → 更多电池设置 → 优化电池使用 → CatlabPing → **不优化**
- 设置 → 应用管理 → CatlabPing → 耗电管理 → **允许后台运行**

### 华为 (HarmonyOS)
- 设置 → 电池 → 启动管理 → CatlabPing → **手动管理**（允许自启动、后台活动、关联启动）
